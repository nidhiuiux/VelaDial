# Concept 01: Minimal Thermostat

Inspired by classic smart thermostats, this UI focuses on a single large numerical value in the center of the screen. A thin amber arc around the edge provides a secondary visual indicator of the brightness level.

**Key Features:**
- Huge, highly readable center text
- Thin edge arc for quick visual reference
- Small status indicators (Room name, ON/OFF state)
- Page navigation dots at the bottom

**Why it works for VelaDial:**
It is extremely legible from a distance (e.g., from the bed looking at the door). The large text ensures there is no ambiguity about the current brightness level.
