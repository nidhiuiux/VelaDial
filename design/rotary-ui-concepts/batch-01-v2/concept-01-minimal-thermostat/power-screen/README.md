# Concept 01: Minimal Thermostat (Power Screen) - Model Comparison

This directory contains 6 distinct design directions for the "Power Screen" of the Minimal Thermostat concept.
Each of the 6 directions has been rendered using all 4 available AI image generation models to allow for direct quality comparison.

## Design Directions:
1. **Option 01:** Glowing Power Symbol (Classic)
2. **Option 02:** Thin-Line Minimal (Apple-style)
3. **Option 03:** Pulsing Orb (Organic/Breathing)
4. **Option 04:** Iris Aperture (Mechanical)
5. **Option 05:** Eclipse Corona (Astronomical)
6. **Option 06:** Radar Sweep (Tech/Precision)

## Models Compared:
- `nano-banana-pro` (2048x2048)
- `gpt-image-2` (1248x1248)
- `nano-banana-2` (2048x2048)
- `default` (1248x1248)
