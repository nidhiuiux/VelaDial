# Concept 08: Three-Screen Tab Carousel

This is the standard approach defined in the App Flow document, featuring visible navigation hints to show the user where they are in the 3-page layout.

**Key Features:**
- Standard brightness arc and percentage
- Prominent navigation dots at the bottom
- Subtle "<" and ">" swipe hints on the edges

**Why it works for VelaDial:**
It uses a familiar mobile OS pattern. The user never gets lost because the navigation dots clearly indicate the current page (Power, Brightness, or Presets) and how to get to the others.
