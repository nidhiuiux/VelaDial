# Concept 06: Night Mode Ultra-Minimal

Designed specifically for zero light pollution, this UI uses only thin wireframe outlines and dark, dimmed colors. It avoids solid fills entirely.

**Key Features:**
- Very dim amber and dark gray colors
- Thin wireframe power icon
- Dimmed percentage text
- Subtle "NIGHT" label

**Why it works for VelaDial:**
It is perfect for sensitive sleepers. When the screen wakes from its 10% idle state, it doesn't blind the user. It provides just enough information to be usable in a pitch-black room.
