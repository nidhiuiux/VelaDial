# Concept 04: Single-Page Simple Mode

This concept abandons the multi-page swipe model entirely. It combines power toggle and brightness adjustment onto a single, unified screen.

**Key Features:**
- Center power icon (tap to toggle)
- Brightness percentage below the icon
- Thin brightness arc around the edge (rotate to adjust)
- No swipe navigation required

**Why it works for VelaDial:**
It offers the lowest cognitive load. The user never has to wonder which page they are on. Pressing the screen toggles power; turning the knob adjusts brightness. It sacrifices color presets for ultimate simplicity.
