# Concept 09: Brightness-First UI

Assuming the user mostly wants to dim the lights, this UI makes the brightness arc the dominant element and shrinks the power toggle.

**Key Features:**
- Very thick, prominent brightness arc
- Large center percentage
- Small power button tucked below the percentage
- "rotate to dim" helper text

**Why it works for VelaDial:**
It optimizes for the rotary knob's primary function. It removes the need to swipe to a dedicated brightness page, making dimming a zero-friction action.
