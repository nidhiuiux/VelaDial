# Concept 07: Text-First Utility UI

This concept abandons icons and abstract arcs in favor of clear, unambiguous text labels stacked vertically.

**Key Features:**
- Stacked text layout
- Large, colored state text ("ON" in green, "OFF" in red/gray)
- Brightness percentage in amber
- Active preset name spelled out clearly

**Why it works for VelaDial:**
There is zero ambiguity. The user doesn't have to guess what an icon means or interpret an arc's fill level. It is highly utilitarian and readable.
